# 🌸 Garden Crush

> A 2D game developed in C using the Raylib graphics library.

---

## 📖 About the Project

Garden Crush is a 2D match-3 game built from scratch using the C programming language and the Raylib library. It was developed as a team project with a focus on low-level game development concepts, without using a game engine.

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| **C** | Core programming language |
| **Raylib** | Graphics, input, and audio library |
| **Visual Studio 2022** | IDE and build environment |

---

## 🚀 Getting Started

### Prerequisites

- [Visual Studio 2022](https://visualstudio.microsoft.com/)
- [Raylib](https://www.raylib.com/) (v4.x or higher recommended)
- Windows 10/11

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yagmur-polat/garden-crush.git
   ```
2. Open the project in Visual Studio 2022
3. Make sure Raylib is linked correctly in the project properties
4. Build and run (`Ctrl + F5`)

## 🎮 Gameplay

- 2D match-3 puzzle game (similar to Candy Crush mechanics)
- Developed using only C and Raylib — no game engine
- Custom game loop, rendering, and input handling
